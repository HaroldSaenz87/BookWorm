# BookDrop
### Social Book Discovery & Recommendations

BookDrop is a modern mobile platform built with React Native and Expo that allows book lovers to share their favorite reads, discover new titles through a community feed, and manage their own "library" of recommendations.

---

## Key Features

- **Paginated Community Feed** — Browse book recommendations from users globally with efficient infinite scrolling.
- **User Profiles** — View and manage your personal "drops," including the ability to delete old recommendations.
- **Interactive Ratings** — Share books with 1-5 star ratings and detailed captions.
- **Smooth UX** — Custom "flying book" loading animations and pull-to-refresh functionality.
- **Authentication** — Secure login and registration system with persistent user sessions.

---

## 🛠️ Technical Stack

### Frontend
| Technology | Purpose |
|---|---|
| [Expo](https://expo.dev/) (React Native) | Framework |
| Expo Router | File-based navigation |
| [Zustand](https://github.com/pmndrs/zustand) | Global auth and data state |
| `expo-image` | High-performance image caching |
| `AsyncStorage` | Local token persistence |

### Backend
| Technology | Purpose |
|---|---|
| Node.js | Runtime |
| Express.js | Framework |
| MongoDB (Mongoose) | Database |
| Render | Deployment |
